# Quick Commerce Analytics Dashboard

## Project Overview

This project is an end-to-end Quick Commerce Analytics Dashboard designed to analyze operational efficiency, delivery performance, customer behavior, and business growth metrics for a quick-commerce platform similar to Blinkit, Zepto, and Instamart.

The dashboard transforms raw order and delivery data into actionable business insights that support data-driven decision-making.

---

## Business Problem

Quick-commerce companies operate in a highly competitive environment where delivery speed, customer satisfaction, and operational efficiency directly impact profitability.

This project aims to answer:

* How efficiently are orders being delivered?
* Which cities generate the highest revenue?
* What factors contribute to order cancellations?
* When are peak ordering periods?
* How effectively are delivery partners performing?
